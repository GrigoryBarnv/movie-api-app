Movie API Documentation

POST /users

Create a new user.

Method: POST

URL: /users

Body:

{
  "name": "Alice",
  "favoriteMovies": ["Inception", "Matrix"]
}

Response (201):

{
  "id": "056c5554-bde6-42f3-b832-2a3864bfc406",
  "name": "Alice",
  "favoriteMovies": ["Inception", "Matrix"]
}

POST /users/:id/:movieTitle

Add a movie to a user’s favorites.

Method: POST

URL: /users/{id}/{movieTitle}

Path parameters: id (string), movieTitle (string)

Response (200):

"Inception has been added to user 056c5554's array"

DELETE /users/:id/:movieTitle

Remove a movie from a user’s favorites.

Method: DELETE

URL: /users/{id}/{movieTitle}

Response (200):

"Inception has been removed from user 056c5554's array"

DELETE /users/:id

Delete a user by ID.

Method: DELETE

URL: /users/{id}

Response (200):

"user 056c5554 has been removed"

GET /movies/:title

Get movie details by title.

Method: GET

URL: /movies/{title}

Response (200):

{
  "Title": "Inception",
  "Genre": { "Name": "Sci-Fi" },
  "Director": { "Name": "Christopher Nolan" }
}

GET /movies/genre/:genreName

Get details about a genre.

Method: GET

URL: /movies/genre/{genreName}

Response (200):

{ "Name": "Sci-Fi" }

GET /movies/director/:directorName

Get details about a director.

Method: GET

URL: /movies/director/{directorName}

Response (200):

{ "Name": "Christopher Nolan" }
