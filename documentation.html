myFlix API

The myFlix API provides access to information about movies, directors, and genres. Users can register, update their profile, and create a list of favorite movies. This API is part of the full-stack JavaScript project (MERN stack).

Available Features (so far)

  • Get a list of all movies
  • Get data about a single movie by title
  • Get data about a genre by name
  • Get data about a director by name
  • Register as a new user
  • Update user info
  • Add or remove movies from favorites
  • Deregister as a user
