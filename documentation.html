Movie API Documentation

GET /movies

Get a list of all movies.

Method: GET

URL: /movies

Response (200):

[
  { "Title": "Inception", "Genre": { "Name": "Sci-Fi" }, "Director": { "Name": "Christopher Nolan" } },
  { "Title": "The Godfather", "Genre": { "Name": "Crime" }, "Director": { "Name": "Francis Ford Coppola" } }
]

GET /movies/:title

Get movie details by title.

Method: GET

URL: /movies/{title}

Response (200):

{
  "Title": "Inception",
  "Genre": { "Name": "Sci-Fi" },
  "Director": { "Name": "Christopher Nolan" }
}

GET /movies/genre/:genreName

Get details about a genre.

Method: GET

URL: /movies/genre/{genreName}

Response (200):

{ "Name": "Sci-Fi", "Description": "Science fiction films explore futuristic concepts..." }

GET /movies/director/:directorName

Get details about a director.

Method: GET

URL: /movies/director/{directorName}

Response (200):

{ "Name": "Christopher Nolan", "Bio": "British-American filmmaker...", "Birth": 1970 }

POST /users

Create a new user.

Method: POST

URL: /users

Body:

{
  "Username": "Alice",
  "Password": "hashedpw",
  "Email": "alice@example.com",
  "Birthday": "1990-05-12"
}

Response (201):

{
  "id": "056c5554-bde6-42f3-b832-2a3864bfc406",
  "Username": "Alice"
}

GET /users

Get a list of all users.

Method: GET

URL: /users

Response (200):

[
  { "Username": "Alice", "Email": "alice@example.com" },
  { "Username": "Mark", "Email": "mark@example.com" }
]

PUT /users/:id

Update user information.

Method: PUT

URL: /users/{id}

Body:

{ "Email": "newalice@example.com" }

Response (200):

{ "message": "User updated successfully" }

POST /users/:id/:movieTitle

Add a movie to a user’s favorites.

Method: POST

URL: /users/{id}/{movieTitle}

Response (200):

"Inception has been added to user 056c5554's favorites"

DELETE /users/:id/:movieTitle

Remove a movie from a user’s favorites.

Method: DELETE

URL: /users/{id}/{movieTitle}

Response (200):

"Inception has been removed from user 056c5554's favorites"

DELETE /users/:id

Delete a user by ID.

Method: DELETE

URL: /users/{id}

Response (200):

"user 056c5554 has been removed"
